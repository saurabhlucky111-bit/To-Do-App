# 📝 To-Do App

A simple and responsive To-Do application built using HTML, CSS, and JavaScript.

## 📸 Screenshot

![To-Do App](screenshot.png)

## 🚀 Features

- ✅ Add new tasks
- ✅ Delete tasks
- ✅ Mark tasks as completed
- ✅ Persistent storage using Local Storage
- ✅ Tasks remain after page refresh
- ✅ Live statistics:
  - Total Tasks
  - Completed Tasks
  - Pending Tasks

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Local Storage
